# Print graphs to terminal

Prints an igraph object to terminal (different than the standard igraph
method)

## Usage

``` r
# S3 method for class 'igraph'
str(object, ...)
```

## Arguments

- object:

  An igraph object

- ...:

  additional arguments to print (ignored)

## Value

str does not return anything. The obvious side effect is output to the
terminal.
